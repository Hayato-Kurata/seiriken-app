## 基本情報
- **イベント**: ニコニコ超会議 2026
- **ブース**: クリエイタークロス No.0334
- **日程**: 4/25(土) 10:00〜18:00 / 4/26(日) 10:00〜17:00
- **席数**: 9席
- **制作時間**: 約20分/人

---

## 事前準備（当日朝）

### 1. サーバー起動
```
cd seiriken-app
npm start
```

### 2. スタッフ画面にログイン
- ブラウザで `http://（公開URL）/admin.html` を開く
- パスワード: `yubi2026`

### 3. QRコード準備
- スタッフ画面の下にある「来場者用QRコード」セクション
- 公開URLを入力して「生成」ボタン
- 表示されたQRコードを印刷してブースに掲示

---

## 運用の流れ

### 来場者の整理券取得
1. 来場者がQRコードをスマホで読み取る
2. 参加人数（1〜6名）を選択
3. 「整理券を取得する」を押す
4. 番号が発行される（同じ端末からは1日1回のみ）

### スタッフの操作（3つのボタンだけ）

| ボタン | いつ押す？ |
|--------|-----------|
| **📢 次の番号を呼び出す** | 席が空いていて、次の人を呼びたい時 |
| **🪑 席が空いた（1席）** | 1人（1組）が完了した時 → 自動で次を呼び出します |
| **🪑🪑 2席同時に空いた** | 2組が同時に終わった時 → 2人分自動呼び出し |
| **🪑🪑🪑 3席同時に空いた** | 3組が同時に終わった時 → 3人分自動呼び出し |

### 基本パターン
```
開場 → 「📢 次の番号を呼び出す」を9回押して9席埋める
  ↓
1組完了 → 「🪑 席が空いた（1席）」を押す → 次の人が自動で呼ばれる
  ↓
（以降繰り返し）
```

---

## 来場者側の画面

### 整理券取得前
- 空き席数、待ち人数、待ち時間目安が表示される
- 自動更新（10秒ごと）

### 整理券取得後
- 自分の番号、前に何人待ちかが表示される
- 自動更新（5秒ごと）
- **順番が来たらブラウザ通知が届く**
  - ※来場者がページを開いたままにしている必要あり

---

## カップル・家族の対応
- 来場者が整理券取得時に人数を選択する（1〜6名）
- 1組1番号で管理（番号は1つ）
- 席が足りない場合は口頭で「少々お待ちください」と案内

---

## よくあるケース

### 呼び出したけど来ない
→ もう一度「📢 次の番号を呼び出す」を押して次の人を呼ぶ

### 2席以上同時に空いた
→ 「🪑🪑 2席同時に空いた」または「🪑🪑🪑 3席同時」ボタンを使う
→ 4席以上の場合は「1席」ボタンを必要回数押す

### 整理券を取ったのに番号が表示されない（来場者から問い合わせ）
→ ブラウザをリロードしてもらう
→ それでもダメなら、localStorageをクリアしてもう一度取得してもらう

### サーバーが落ちた
→ `npm start` で再起動
→ ※メモリ上のデータは消えるので番号はリセットされる

---

## 日替わりリセット
- 番号は日付ごとに自動リセット（25日と26日で別管理）
- 特に操作は不要

---

## 注意事項
- スタッフ画面のURLは来場者に教えないこと
- パスワードは当日スタッフ間で共有
- サーバーを再起動するとデータがリセットされる（本番中は再起動しない）
- 来場者の通知は、ページを閉じると届かなくなる
